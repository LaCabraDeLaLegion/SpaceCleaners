Movimiento del personaje

Para desplazarte por la pantalla puedes utilizar las flechas o las teclas A, W, S, D. Es posible desplazarse en cualquier dirección, incluidas las diagonales.

Disparar

Para disparar un arma utilizar la barra espaciadora.
Para disparar una medicina utilizar la tecla C.

ATENCIÓN: si disparas a un humano con un arma mutará y se convertirá en un virus
ATENCIÓN: si disparas a un virus con una medicina mutará, posiblemente subiendo de nivel, restaurando su vida

Consumibles

Para usar un escudo utiliza la tecla 1
Para utilizar una poción utiliza la tecla 2
Para lanzar una bomba utiliza la tecla 3
